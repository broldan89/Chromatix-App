# 📍 Roadmap de Chromatix: El Sistema Operativo del Color

Este es el plan de ejecución para transformar los datos maestros en una solución integral para el salón profesional.

---

### Fase 1: Cimentación (Completado ✅)
- [x] Definición de lógica de colorimetría (Yellow Professional).
- [x] Estructuración de datos maestros en CSV (Niveles, Reflejos, Ratios, Tiempos).
- [x] Creación del repositorio oficial en GitHub con documentación técnica.
- [x] Prototipado inicial de UI/UX (Figma Make).

### Fase 2: Desarrollo del MVP (Próximamente 🛠️)
- [ ] **Entorno de Desarrollo:** Configuración inicial en React Native / Flutter para despliegue multiplataforma.
- [ ] **Motor de Cálculo:** Programación del algoritmo que cruza las 4 tablas base para generar fórmulas.
- [ ] **Interfaz de Diagnóstico:** Selector visual de Nivel Natural (1-10) y Objetivo deseado.
- [ ] **Módulo de Resultados:** Visualización de mezcla exacta (gramos/ml) y temporizador (Timer) de exposición.

### Fase 3: Evolución Técnica & Cuidado (Fase Pro 🚀)
- [ ] **Módulo de Aditivos (Bond Hero):** Sugerencia automática de protectores según la potencia del oxidante (>= 30 Vol) o procesos de decoloración.
- [ ] **Módulo de Decoloración:** Lógica para saltos de +4 niveles con control de etapas de fondo y fondos de aclarado (8, 9, 10).
- [ ] **Historial de Clientes:** Base de datos en la nube para guardar diagnósticos, fotos del "antes/después" y fórmulas exitosas.
- [ ] **Sistema Multimarca:** Motor agnóstico capaz de alternar entre diferentes cartas de color mediante carga de perfiles CSV.

### Fase 4: Ecosistema Comercial & Supply Chain (Monetización 💰)
- [ ] **Gestión de Inventario en Tiempo Real:** Resta automática de stock (tinte, oxidante, aditivos) tras cada servicio confirmado.
- [ ] **Smart Shopping List:** Generación de listas de reposición basadas en el consumo real y alertas de stock bajo.
- [ ] **Conector de Distribución:** Integración para enviar pedidos de stock faltante directamente a la distribuidora (WhatsApp/API/Email).
- [ ] **Dashboard de Rentabilidad:** Reporte analítico de costos por servicio para ayudar al profesional a optimizar sus márgenes de ganancia.

---

## 📈 Visión a Largo Plazo
Chromatix aspira a ser el estándar de la industria donde la distribuidora, el salón y el colorista convergen en una plataforma que elimina el error humano y garantiza la rentabilidad del negocio.
