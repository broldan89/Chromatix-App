# 📍 Roadmap de Chromatix

Este es el plan de ejecución para transformar estos datos en una aplicación funcional.

### Fase 1: Cimentación (Completado ✅)
- [x] Definición de lógica de colorimetría (Yellow Professional).
- [x] Estructuración de datos maestros en CSV.
- [x] Creación del repositorio oficial en GitHub.
- [x] Prototipado inicial de UI/UX (Figma).

### Fase 2: Desarrollo del MVP (Próximamente 🛠️)
- [ ] Configuración del entorno de desarrollo (React Native / Flutter).
- [ ] Programación del motor de cálculo de mezclas (Lógica de los CSV).
- [ ] Creación de la interfaz de selección de niveles (1-10).
- [ ] Implementación del temporizador (Timer) de exposición.

### Fase 3: Funciones Pro (Futuro 🚀)
- [ ] Historial de clientes con base de datos en la nube.
- [ ] Gestión de inventario de tintes.
- [ ] Soporte para múltiples marcas (Sistema Multitarea).
